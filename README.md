### 개발 환경

- Language : Kotlin
- minSdk : 23
- targetSdk : 31

### 메인 페이지

- 카테고리 API를 사용하여 가져온 카테고리 중 3개 이상의 카테고리를 선정하여 탭을 구성
- 각 탭 클릭 시 그 카테고리에 해당하는 방송 리스트를 보여줌
- 페이징 기능으로 다음 페이지 리스트가 하단에 추가된다.
- 방송 클릭 시 해당 방송의 상세 페이지로 이동

### 상세 페이지

- 클릭한 해당 방송 정보를 상세히 확인 가능
